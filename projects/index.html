Machi Koro

Turn based board game implemented in Ocaml for the terminal.

What it looks like.

  • Explore Repository

What it is.

Machi Koro is a city-building board game published in 2012 by Grounding Inc. and designed by Masao Suganuma.

How we did it.

This was a month long project for one of my favorite programming classes. We were tasked with creating a system from scratch, with non-trivial features. I was lucky enough to work with two awesome programmers that helped make this happen. To make this happen, we had to

  • — Fully implement all the features of Machi Koro in a functional language.
  • — Create a user-friendly text-based engine with easy to understand commands, game flow, and instructions.
  • — Design all the data types and schemas to store the game state and game saves.

ClickerEmpire

Text based strategy game for android.

What it looks like.

  • Download for Android

What it is.

ClickerEmpire is based on CivCLicker. It takes the base features of CivClicker and condenses them into an app for Android.

How it works.

To maintain a record of all the data being processed, a synchronised SQLite database was used in order to avoid race conditions in the multiple threads that run. Fragments were used to have clickable tabs within the main activity so that a new activity would have to be opened every time a player wanted to access a functionality of the game.

Gameplay.

Playing this game is quite simple. You must expand your kingdom by raiding other kingdoms, but you need to make sure that your subjects have enough food to prevent starvation, that you have enough stables and barracks to house your soldiers, and most importantly, that you don't run out of land while you're expanding.

Plans for the future.

I plan to rebuild this app using the Unity Engine some time in the future to provide a more "game-like" experience.
