Galeria

Simple image hosting done in PHP & SQLite

How it works

The purpose of this project was to create a simple image hosting site that allowed users to upload pictures, and provide tags for them to make the pictures searchable. Only registered users can upload pictures, and only the user who uploaded a picture can delete its tags, or the picture itself.
